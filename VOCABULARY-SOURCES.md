# Vocabulary sourcing

The lesson bank is curated for Sabrina's approximate CEFR A1–A2 level.

Primary references:

- [Oxford 3000 by CEFR level](https://www.oxfordlearnersdictionaries.com/external/pdf/wordlists/oxford-3000-5000/The_Oxford_3000_by_CEFR_level.pdf)
- [Cambridge English A2 Key vocabulary list](https://www.cambridgeenglish.org/images/506886-a2-key-2020-vocabulary-list.pdf)
- [British Council A1–A2 vocabulary topics](https://learnenglish.britishcouncil.org/free-resources/vocabulary/a1-a2)

Words are selected from common A1–A2 vocabulary and grouped into practical themes. Chinese translations, phonetics, and example sentences are written specifically for this site rather than copied from those sources.

The vocabulary lesson sequence advances on each Mode A day. It does not use the calendar day modulo the number of sets, so a set is not repeated until the entire bank has been used.
